摘要

注意 
(由于hexo主题存在太多兼容性问题和坑，已改用Hugo搭建博客)
利用hexo和gitee的 gitee page搭建个人博客"

操作步骤：

1.安装环境依赖

下载最新的nodejs手动安装：https://nodejs.org/en/download/

1
2
3

    wget https://nodejs.org/dist/v20.11.0/node-v20.11.0-linux-x64.tar.xz
    sudo mkdir -p /usr/local/lib/nodejs
    sudo tar -xJvf node-v20.11.0-linux-x64.tar.xz -C /usr/local/lib/nodejs

添加环境变量 用户.bashrc和系统/etc/profile

1
2

    sudo nano /home/zhb/.bashrc
    sudo nano /etc/profile

都要在最后加入下面这行

1
2
3

    export PATH=/usr/local/lib/nodejs/node-v20.11.0-linux-x64/bin:$PATH
    source .bashrc
    source /etc/profile

之后命令行输入

1
2

    node -v  
    # 有显示版本信息即为成功

安装git：

1

    sudo apt-get install git-core

安装cnpm(非必须):

1

    npm install -g cnpm --registry=https://registry.npm.taobao.org

2.安装hexo-cli

1
2
3

    npm install -g hexo-cli
    或者
    cnpm install -g hexo-cli

验证： hexo -v

3.初始化hexo

1
2

    hexo init myblog
    # 这里大概率会出现git clone失败的错误

1

sudo nano /usr/local/lib/nodejs/node-v20.11.0-linux-x64/lib/node_modules/hexo-cli/dist/console/init.js

改一下这个文件init.js,把原来的git url替换为以下：

1

    const GIT_REPO_URL = 'https://gitee.com/naonano/hexo-starter.git'

之后重新执行hexo init

4.安装npm

1
2

    cd myblog
    npm install

5.启动站点

1
2

    hexo g
    hexo server

6.在gitee搭建站点

gitee上创建仓库，仓库名称随意，路径设置为：用户名+.gitee.io 

例如： binbyzhang.gitee.io

安装一个插件hexo-deployer

1

    npm install hexo-deployer-git --save

修改hexo配置文件指定仓库路径

1

    sudo nano _config.yml

添加以下内容：

1
2
3
4

    deploy:
      type: git
      repo: https://gitee.com/naonano/binbyzhang.gitee.io.git
      branch: master

推送站点文件到gitee

1
2
3
4
5

    git config --global user.email "gitee的邮箱"
    git config --global user.name "gitee用户名"
    git config --global user.password "gitee用户密码"
    hexo d
    输入gitee账号和密码

然后开启仓库的 gitee pages服务，勾选强制https服务

7.访问测试

1

    https://naonano.gitee.io/binbyzhang.gitee.io/
