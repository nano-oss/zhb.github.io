下载VMware，并安装：

1
2

https://pan.baidu.com/s/1Hc3yK1oVUY3m3N6ZDgZE3A
提取码：3456

VMware下ubuntu设置共享文件夹：

1
2
3
4

sudo apt update
sudo apt install open-vm-tools-desktop fuse
sudo mkdir /mnt/hgfs
sudo mount -t fuse.vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other

下载一个Linux镜像，推荐ubuntu:

1

https://mirrors-i.tuna.tsinghua.edu.cn/

！最好用2004及以上版本的ubuntu，低版本会因为python版本错误导致安装失败

常规安装虚拟机进行即可 vmware-tools可能安装不上:

1
2

点击安装vmware-tools之后登出用户重新进来一般可以
如果出现两个询问窗口，第二个窗口点击否，也可以解决

换源：

1
2

sudo mv /etc/apt/sources.list /etc/apt/sources.list.bak
sudo nano /etc/apt/sources.list

写入以下内容覆盖原文件，根据系统选择：

  • Ubuntu 16.04 (LTS)代号为xenial。
  • Ubuntu 18.04 (LTS)代号为bionic。
  • Ubuntu 20.04 (LTS)代号为focal。

输入lsb_release -a可以查看

更新：

1
2

sudo apt-get update
sudo apt-get upgrade

安装常规软件：

1

sudo apt-get install git gcc make g++ tree nano vim openssh-server python3.7 net-tools dfu-util

安装klipper

1
2

git clone https://gitee.com/miroky/kiauh.git
sudo ./kiauh

查看python3版本：

1

python3 -V

klipper最低支持py3.7, 不是3.7以上版本的话需要更新: 查看python各版本位置

1

whereis python

更改python默认版本指向

1
2
3

sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.7 1
取消指向
sudo update-alternatives --remove python3 /usr/bin/python3.7
